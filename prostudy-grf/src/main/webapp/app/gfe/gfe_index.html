Google Classroom Dashboard

Following are the key features of Google Classroom Dashboard :

  • Google Classroom Dashboard show the all courses availble in your school. You can add, edit and view all courses.
  • From Google Classroom Dashboard you can add and view the teachers associated to particular courses and you can delete teacher from courses.
  • From Google Classroom Dashboard you can add multiple courses and teacher, co teachers by uploading data using CSV.
  • From Google Classroom Dashboard you can archive and delete one or more courses.
  • From Google Classroom Dashboard you can add and view the student associated to particular courses and you can delete student from courses.
  • From Google Classroom Dashboard student can goto to classroom see the assignment given to them by teachers.
  • From Google Classroom Dashboard student can see the course calendar which shows the assignment and work in current week.
  • From Google Classroom Dashboard teacher can make annoucement regarding exam, submissions and create questions set for students.
  • From Google Classroom Dashboard teacher can upload study material for student regarding course.
  • From Google Classroom Dashboard teacher you can filter list by searching course name or teacher.
  • From Google Classroom Dashboard teacher you can filter list by course state like whether course state is active or archive.

Google User Directory
  • Google User Directory shows the all users list.
  • In Google User Directory you can add, edit, view and delete user from user list.
  • In Google User Directory you can add multiple users by uploading csv data.
  • In Google User Directory you can filter list by searching user.
